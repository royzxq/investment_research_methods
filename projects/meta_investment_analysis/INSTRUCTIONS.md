# 信息来源要求

你需要围绕调研时点，主动调研并综合以下公开信息：

- 宏观经济与流动性环境

- 政策与监管环境

- 中观产业景气与主线变化

- 资金行为与市场风格偏好

- 估值与风险偏好状态

- 海外市场与外部叙事对中国资产的映射影响



要求：

- 避免被单日噪音、短期情绪或单条新闻误导

- 不追求面面俱到，重点识别真正影响研究框架的核心因素

- 区分“表面热闹”和“研究意义上的重要变化”

# 分析原则

- 你的任务不是复述新闻，而是提炼会影响研究方法的核心变量

- 优先识别：会驱动轮动、收益来源迁移、估值体系切换、研究优先级变化的因素

- 必须区分“阶段性噪音”与“足以影响研究框架的变化”

- 不直接输出个股推荐

- 不直接输出个股买卖建议

- 本次任务只输出“当前市场状态的结构化调研结果”



# 分析步骤



## 第一步：识别当前市场主要矛盾

要求：

- 只能用一句话概括

- 必须是能驱动轮动、收益来源迁移或估值体系变化的矛盾

- 不能写成新闻标题式总结

- 不能停留在“市场震荡/分化”这种空泛描述



输出：

- 当前主要矛盾

- 谁在解决这个矛盾

- 谁在被这个矛盾压制



## 第二步：判断当前主导收益来源

从以下机制中选出 1–2 个最主导的：

- 估值修复

- 盈利兑现

- 风险偏好抬升

- 流动性宽松

- 供给出清

- 政策重估

- 海外映射 / 外部叙事传导



要求：

- 必须结合调研时点附近的公开信息判断

- 说明为什么它们是主导项

- 说明为什么其他项不是当前主导



输出：

- 主导收益来源

- 证据

- 非主导项说明



## 第三步：判断当前关键约束

从以下因素中识别 1–3 个最重要的：

- 需求不足

- 风险偏好偏弱

- 拥挤交易

- 竞争恶化

- 再融资压力

- 资本开支过热

- 海外扰动

- 政策不确定性

- 产业链价格压力

- 基本面兑现难度上升



要求：

- 只保留真正会影响研究框架的约束

- 不要把所有风险都列上去

- 必须说明这些约束会压制哪些资产、利好哪些资产



输出：

- 当前关键约束

- 哪些资产最受其影响

- 哪些资产反而因其受益



## 第四步：判断当前优先研究机制

对以下机制分别判断优先级：高 / 中 / 低

- 正反馈强化型

- 负反馈修复型

- 周期反转型

- 稳态现金流型

- 事件驱动重估型



要求：

- 给出每类机制优先级

- 解释原因

- 说明当前最容易误判哪类机制



## 第五步：识别当前风险点

输出：

- 当前最脆弱的叙事

- 当前最不能犯的错误

- 当前最需要前置验证的事项（3项以内）



要求：

- “最脆弱的叙事”是指最容易被证伪、但市场容易高估的逻辑

- “最不能犯的错误”是指当前阶段最致命的研究错误，不是泛泛而谈

- “前置验证项”必须是真正会改变研究结论的关键检查项
